# Agent skills

You will find Agent skills at the Python tools for the Polylith Architecture [GitHub page](https://github.com/DavidVujic/python-polylith).

The skills will show your agent how to use the Python tool, and will add knowledge about the Polylith Architecture in general.

This can be helpful for teams that practice _Agentic Engineering_. The agents will be able to determine how to use the CLI in general, create bricks, validate the workspace, getting information about the projects, interfaces and dependencies.

 > :material-information: There's a tool called _library-skills_ (developed by the creator of FastAPI) that can extract any skills added by the installed libraries in a repo. You can add the Polylith skills to your Monorepo with that tool.

Example, using library-skills:
```shell
uvx library-skills
```
Or just copy the `.agents` folder from the Python tools for the Polylith Architecture GitHub page.
